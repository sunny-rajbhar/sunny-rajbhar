# GitHub Push Steps:
first create github clone to local
1. git clone https://github.com/sunny-rajbhar/django-blog.git

then move to cloned repo
2. cd django-blog/

3. git add .

verify user 
4. Run
  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"

commit with comments
5. git commit -m "Adding project to github repo"

upload to github 
6. git push origin
